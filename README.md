## About Me

I am a Computer Science graduate and AI builder focused on **agentic systems, applied machine learning, RAG, NLP, and full-stack AI products**.

I like building systems where the model is only one part of the product: real tools, useful memory, clean interfaces, measurable behavior, and deployment paths that someone can actually run.

```text
Current direction:
- Agentic workflows with real tool use, evaluation, and guardrails
- Retrieval systems that preserve context and improve reliability
- Local-first assistants that respect privacy and cost constraints
- Healthcare, productivity, and research-focused AI applications
- Full-stack interfaces that make complex AI systems easy to use
```

## Featured Projects

### [AIA - Local Multi-Agent Personal Assistant](https://github.com/attaquarks/ai-assistant-system)

Local-first personal assistant with independent business and health agents. Built around Docker Compose, FastAPI, Redis Streams, PostgreSQL/pgvector, Ollama, and WhatsApp gateway integration.

`Python` `FastAPI` `Docker` `Ollama` `Redis` `PostgreSQL` `WhatsApp`

---

### [Supply Chain Intelligence Agent](https://github.com/attaquarks/Capstone)

Industrial agentic AI system built with LangGraph for inventory analysis, supplier intelligence, reorder-risk scoring, procurement email drafting, guardrails, evaluation, CI, and drift monitoring.

`Python` `LangGraph` `RAG` `FastAPI` `Streamlit` `Docker` `ChromaDB`

---

### [3AM Talks - AI Therapy System](https://github.com/attaquarks/3am-talks)

AI-powered therapy and counseling system using personality-aware responses, RAG, user profiles, conversation memory, and evidence-based therapeutic technique retrieval.

`Python` `RAG` `Supabase` `Pinecone` `Ollama` `NLP`

---

### [NeuroReport](https://github.com/attaquarks/NeuroReport)

Medical imaging VQA and report-generation pipeline for MRI scans using vision encoders, slice aggregation, vision-language bridging, biomedical language models, and PyTorch Lightning training workflows.

`Python` `PyTorch` `Transformers` `Medical AI` `VQA` `Report Generation`

---

### [SymptoLens](https://github.com/attaquarks/SymptoLens)

TypeScript web application for AI-assisted symptom analysis with structured medical reasoning, React UI, Express backend, PostgreSQL, authentication, and real-time features.

`TypeScript` `React` `Express` `PostgreSQL` `Tailwind CSS` `Medical AI`
